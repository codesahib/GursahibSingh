Machine Learning Project

Title - Stock Price Predictor

Introduction

The project demonstrates basics of Machine Learning via implementation of a model which can help predict the stock prices.

Libraries Used

  • Pandas
  • Pandas Datareader
  • Numpy
  • Datetime
  • Matplotlib
  • Sklearn
  • Quandl

Methodology

  • Initially start date and end date for a period of consideration is defined using Datetime
  • Then, datareader is used to get data related to particular stock
  • The data frame is pre-processed using numpy
  • Last n days data is separated and saved as forecast. This constitutes the resultant set
  • Data is split between training and testing data.
  • Linear Regression model is used on the training and testing data
